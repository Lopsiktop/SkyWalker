# SkyWalker

Скайволкер - это проект, в котором вы можете арендовать чей-то транспорт (по часам, по дням) или добавить свой транспорт чтобы его могли арендовать другие пользователи.

Если вы решили доработать проект, то вам нужно следать следующее:

* В Visual Studio Средства -> Диспетчер пакетов NuGet -> Консоль диспетчера пакетов
* В консоль ввести "update-database"
